## Install  
#### Requirements  
`node` environment is required before install.   
It is recommended to use [cnpm](http://npm.taobao.org/) instead of npm in Mainland China.  
#### Download
```shell
git clone https://github.com/stkevintan/TaskManager/
```
#### Install Dependances
```shell
npm i
```
#### Build
```shell
npm run build
```
## Usage
```shell
npm start
```
